# Show Me — 单页网站设计

文件: `show-me-design.pen`

## 设计概览

一个温暖、克制、叙事驱动的个人作品展示网站。单页滚动式设计，包含六个核心区域。

### 设计理念
- **Less but better** — 像 Dieter Rams 一样剥离一切多余
- **温暖的未来感** — 大地色系、柔和光感、有肌理的留白
- **内容会生长** — 可扩展的内容容器，新增内容不需要重新装修

### 色彩系统
```
背景色:   #FAF8F5（暖白，像未漂白的纸）
主文字色:  #2C2C2C（深炭灰）
次文字色:  #7A7268（暖灰）
强调色:   #C4956A（柔和琥珀）
辅助色:   #8B9E82（鼠尾草绿）
卡片背景:  #FFFFFF（纯白）
```

### 页面结构

#### 1. Hero（首屏）
- 极简导航栏
- 一句话定位："构建工具，记录思考。"
- 身份描述：帆软平台工程师 · 独立创造者 · AI 原生开发者
- 向下滚动提示

#### 2. About（关于）
- 两段叙事式自我介绍
- 技能标签云：Go / React / AI Agent / RAG / 分布式系统 / TypeScript

#### 3. Works（作品）
- 筛选标签：全部 / 工具 / 平台 / AI
- 四张作品卡片（2x2 网格）
  - Eden — 基于 Obsidian 的博客系统
  - Northstar — 零售经济数据分析工具
  - Confluence MCP — AI 知识库连接器
  - 帆软平台缓存体系 — 三级缓存架构

#### 4. Thinking（思考）
- 主题分类：工程哲学 / AI 实践 / 职业成长 / 生活观察
- 三篇文章列表，带分隔线

#### 5. Journey（旅程）
- 横向时间线：2019 → 2021 → 2023 → 2025
- 四个阶段的关键节点

#### 6. Connect（联系）
- 温暖的收尾语句
- 联系方式：GitHub / Email / Twitter
- 底部标语："Less but better."

### 交互设计
- 平滑滚动锚点导航
- 卡片悬停微动效
- 文字渐显动画
- 克制但不冰冷的氛围

## 详情页设计

### 1. 文章详情页
点击 Thinking 区域的文章后进入的阅读页面：
- **返回导航**：返回思考列表
- **文章头部**：分类标签 + 大标题 + 发布日期
- **分隔线**：优雅的视觉分隔
- **正文区域**：舒适的阅读宽度（720px）
- **引用块**：暖米色背景，斜体显示
- **列表样式**：简洁的项目符号
- **底部留白**：足够的呼吸空间

### 2. 作品详情页
点击 Works 区域的卡片后进入的项目详情页：
- **返回导航**：返回作品列表
- **作品头部**：
  - 标签行（类型标签 + 状态标签）
  - 大标题（Eden）
  - 一句话描述
  - 操作按钮（查看源码 / 在线体验）
- **两栏布局**：
  - **左侧**：背景与动机、方案与实现的叙事内容
  - **右侧**：信息卡片（技术栈、数据、收获）
- **侧边卡片**：白色背景，圆角，包含技术细节

### 3. 时间线节点详情页
点击 Journey 区域的时间节点后进入的阶段详情页：
- **返回导航**：返回旅程主页面
- **年份大标题**：琥珀色大号数字（如 2023）
- **阶段标题**：该阶段的主题（如 独立创造）
- **一句话总结**：概括这个阶段的核心变化
- **关键事件列表**：
  - 左侧：具体时间点（2023.03）
  - 右侧：事件标题 + 描述
  - 三个标志性事件
- **感悟卡片**：白色卡片，斜体引用，记录当时的思考

## 管理后台设计

### 1. 登录页 (/admin/login)
- 居中布局，暖白背景
- Playfair Display 品牌标题 "Show Me" + "管理后台" 副标题
- 用户名/密码输入框（白底、细边框、圆角 8px）
- 深色登录按钮（#2C2C2C）

### 2. 管理看板 (/admin)
只读看板，左侧导航 + 右侧内容区：

**左侧导航栏**（200px 宽，白底）：
- 导航分组：看板总览、作品、文章、旅程、个人信息、联系方式、站点设置
- 工具分组：简历导出
- 当前页高亮（暖灰底 + 琥珀色图标）
- Lucide 图标 + 文字

**右侧内容区**（看板总览页）：
- 概览行：4 张统计卡片（作品数、文章数、阶段数、联系方式数），Playfair Display 大号数字
- 内容列表行：左右分栏
  - 左：作品列表（标题 + 类型标签 + 推荐星标 + 状态标签）
  - 右：文章列表（标题 + 主题 + 发布状态标签）
- 旅程时间线：横向展示年份 → 阶段标题 → 事件数，箭头连接
- 底部行：左右分栏（个人信息 + 联系方式）
- 背景色 #F5F2EE，卡片白底 + 极淡阴影 + 12px 圆角

### 3. 简历导出 (/admin/resume)
基于站点数据自动生成简历，支持预览和 PDF 导出：
- **顶部操作栏**：页面标题 + 刷新预览按钮 + 导出 PDF 按钮（深色主按钮）
- **A4 预览区**：居中展示 595×842 的模拟纸张（白底 + 投影）
  - 简历头部：姓名（Playfair Display）+ 联系信息（右对齐）
  - 关于：个人简介段落
  - 职业经历：职位 + 公司 + 时间 + 描述
  - 代表作品：项目名 + 技术栈 + 描述
  - 技术能力：标签式技能列表
- 所有内容从数据库读取，自动填充

## 页面布局总览

```
前端展示：
  主页面 (1280px)           x=0
  文章详情 (1280px)         x=1320
  作品详情 (1280px)         x=2640
  时间线详情 (1280px)       x=3960

管理后台：
  登录页 (1280px)           x=5280
  管理看板 (1280px)         x=6600
  简历导出 (1280px)         x=7920
```

所有前端页面保持一致的：
- 暖白色背景 #FAF8F5
- 内容宽度 680px（文章）/ 1040px（作品/卡片）
- Playfair Display 标题 + Inter 正文
- 琥珀色 #C4956A 作为强调色

管理后台页面：
- 微暖灰背景 #F5F2EE
- 白色卡片 + 极淡阴影
- 紧凑信息密度，无编辑功能
